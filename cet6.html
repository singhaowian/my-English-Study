翻译长句拆分方法

核心规则：中文多语义叠加，英文坚持一个语义对应一个短句

步骤：1. 划分语义块 2. 拆解短句 3. 使用 and / so / to / which 连接

例句：洋山港是上海航运中心的重要组成部分，是中国首个深水港。
翻译：Yangshan Port is an important part of Shanghai's shipping center. It is China's first deep-water port.
